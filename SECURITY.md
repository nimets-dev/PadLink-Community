# Security Policy

## Reporting a vulnerability

If you discover a security or privacy issue in PadLink, **please do not open a public issue.**
Instead, report it privately:

- **Email:** [support@nimets.com](mailto:support@nimets.com)
- **Telegram:** [@nimets_support](https://t.me/nimets_support)

Please include:

- A description of the issue and its potential impact.
- Steps to reproduce (proof of concept if possible).
- The PadLink version and your OS version.

We'll acknowledge your report as soon as we can and keep you updated on the fix. Thank you for
helping keep PadLink users safe.

## Supported versions

Only the **latest released version** of PadLink receives security updates. Please update to the
newest version before reporting.
